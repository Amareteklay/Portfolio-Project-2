# Rock Paper Scissors Spock Lizard Game

## Rules

* Paper beats Rock
* Rock beats Scissors
* Scissors beat Paper
* Lizard beats Paper
* Spock beats Rock
* Rock beats Lizard
* Lizard beats Spock
* Scissors beat Lizard
* Paper beats Spock
* Spock beats Scissors
